# The Next Web Replica

> A clone of the TNW homepage.

![screenshot](./assets/tnw_screenshot.png)

A project to practice HTML, CSS and responsive design.

## Live Demo

[Live Demo Link](https://condescending-saha-4e8669.netlify.app/)

## Built With

- HTML5
- CSS3

## Getting Started

- Clone the project in to your computer using `git clone https://github.com/m15e/the-next-web-replica.git` 
- `cd` into the `the-next-web-replica` folder
- Open `index.html` in the browser
